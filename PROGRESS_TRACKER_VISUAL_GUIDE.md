### Hover Effects
```

┌─────────┐      ┌─────────┐
│  Card   │  →   │  Card   │  (↑ lifted)
└─────────┘      └─────────┘  (⚡ shadow)
```

### Progress Bars
```
Animation: Left to right fill
Duration: 0.8s
Easing: ease-in-out

Empty:  [□□□□□□□□□□]

```

### Buttons
```

         └─ shadow increases
```

```
1. Page Loads
   ↓
2. Show Loading Spinner
   ↓
3. Fetch APIs (parallel)
   ├─ Streak Stats    ⚡
   ├─ Streak History  ⚡
   ├─ Practice Stats  ⚡
   └─ Learning Data   ⚡
   ↓
4. Process & Calculate
   ↓
5. Animate In (fade + slide up)
   ↓
6. Interactive & Ready
```
